# Claude Code Configuration

This directory contains Claude Code configuration for the Chateau Alaverdi project.

## Available Skills

All skills from `~/.claude/skills/` are available for use in this project:

- ai-photography-prompter-enhanced
- brand-guidelines
- brochure-converter
- canvas-design
- ceo-advisor
- competitive-ux-research
- content-creator
- css-design-system-builder
- design-system-critique
- homepage-content-implementer
- notebooklm
- semantic-html-builder
- ux-accessibility-auditor
- website-photo-analyzer

## Usage

Simply mention the skill name in your conversation with Claude Code to activate it.
